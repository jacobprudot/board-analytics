## Long Description (200-2,500 characters)

```
Board Analytics provides real-time insights and visualizations for your monday.com boards. Get a complete picture of your board health at a glance!

KEY FEATURES:

• Key Metrics Dashboard - See total items, weekly activity, stale items, and overdue tasks instantly

• Status Distribution - Beautiful donut chart showing how items are distributed across statuses

• Group Analysis - Bar chart visualization of items per group

• Activity Timeline - 30-day line chart showing item creation trends

• Issue Detection - Automatically identify overdue and stale items that need attention

• Multiple Views - Four organized tabs (Overview, Distribution, Timeline, Issues)

• Dark Mode Support - Seamlessly adapts to your monday.com theme

• Real-Time Data - Analytics calculated live from your current board state

PERFECT FOR:

- Project managers monitoring overall board health
- Team leads tracking work distribution and bottlenecks
- Executives needing quick status summaries
- Anyone who wants data-driven insights from their boards

No configuration required - just add the view and see your analytics instantly!
```

## Personal Data Use
```
Board Analytics does NOT store any personal or identifiable user data.

The app only accesses board data (item names, dates, column values) through monday.com's API to calculate analytics. All processing occurs in real-time within the browser and monday.com platform.

No data is transmitted to external servers. No databases are used. No cookies store board data.

We do not collect, store, or share any personal information.
```


## Feature Names
```
Board View Feature: "Board Analytics"

This is the main and only feature of the app. It appears as a Board View tab that users can add to any board.

Inside the view, users will find:
- Key Metrics Cards: Total items, created this week, updated this week, stale items, overdue items
- Column Selectors: Dropdowns to choose which status and date columns to analyze
- Tab Navigation: Overview, Distribution, Timeline, Issues
- Overview Tab: Status donut chart + Group bar chart
- Distribution Tab: Detailed status and group bar charts
- Timeline Tab: 30-day item creation line chart
- Issues Tab: Sortable tables for overdue and stale items
- Refresh Button: To reload data on demand

The feature slug in monday-app.json is: "board-analytics-view"
```
